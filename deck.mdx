# Bekk sin MDX-mal

Dette er MDX-malen til Bekk. 

---

## Hva er MDX?

Et fancy nytt presentasjonsrammeverk!

Du kan lage din egen slik:

```bash
npx create-deck min-presentasjon
cd min-presentasjon
npm start
```

[Les mer her](https://jxnblk.com/mdx-deck)

## Overskrift

### Om Bekk

Vi er et norsk selskap med over 400 spesialister innen strategi, design og 
teknologi. Virksomhetene vi jobber med har ambisiøse mål, og gir oss krevende 
utfordringer. 

Vi tror på smart bruk av teknologi, tverrfaglige team og evnen til å tenke 
annerledes. Det gjør at vi kan lage løsninger som utfordrer etablerte sannheter, 
og skape nye muligheter for våre kunder.
